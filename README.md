# DuitOnTrack - AI Finance Coach




## Overview
DuitOnTrack is a personal finance tracking web application designed for students 
to monitor their spending habits and make smarter financial decisions. Powered by 
AI, it provides real-time insights and an interactive dashboard to help users stay 
on top of their financial goals.

> ⚠️ Disclaimer: AI features may be unavailable once the API token limit is reached.



## App deployment :
https://duit-on-track.vercel.app/




## Features
- 📊 Interactive spending dashboard to track and revisit expenditure goals
- 🤖 AI-powered financial insights using Z.AI for smart decision making
- 🔐 Secure user authentication via Firebase Authentication
- 💾 Real-time data storage and management with Firebase Firestore



## Technologies Used
| Technology | Purpose |
|---|---|
| HTML, CSS, JavaScript | Frontend development |
| Firebase Authentication | User login and security |
| Firebase Firestore | Real-time database |
| Z.AI | AI-powered financial insights |
| Vercel | App deployment |
